# Keyboard Shortcuts

`Cmd` on macOS, `Ctrl` on Linux and Windows.

## Canvas

| Shortcut | Action |
|---|---|
| Drag empty canvas | Pan |
| Scroll / pinch | Zoom at cursor |
| Double-click empty canvas | Create a node of the last-used kind |
| Drag a node | Move it (snaps to the grid when snap is on) |
| Drag a group | Move it together with its children |
| Click a port dot, then a target node | Create an edge |
| <kbd>Escape</kbd> | Cancel an in-progress connection |

## Selection and editing

| Shortcut | Action |
|---|---|
| Click | Select node or edge |
| Shift-click | Add to or remove from the selection |
| Double-click a node | Open the Inspector |
| Double-click a node title | Rename inline |
| Double-click a note card | Edit the note text inline |
| Double-click an edge | Edit the edge label inline |
| <kbd>Delete</kbd> / <kbd>Backspace</kbd> | Delete selection |
| <kbd>Cmd</kbd>+<kbd>Z</kbd> | Undo |
| <kbd>Cmd</kbd>+<kbd>Shift</kbd>+<kbd>Z</kbd> or <kbd>Cmd</kbd>+<kbd>Y</kbd> | Redo |

## Operating

| Shortcut | Action |
|---|---|
| <kbd>Cmd</kbd>+<kbd>Enter</kbd> | Open an SSH or kubectl shell on the selected node |
| <kbd>g</kbd> | Toggle snap-to-grid |
| <kbd>n</kbd> | Toggle natural scroll direction |

## Application

| Shortcut | Action |
|---|---|
| <kbd>Cmd</kbd>+<kbd>=</kbd> / <kbd>Cmd</kbd>+<kbd>-</kbd> | UI scale up / down |
| <kbd>Cmd</kbd>+<kbd>0</kbd> | Reset UI scale |

Shortcuts stay quiet while you are typing in an input field or inside a
terminal; keystrokes there belong to the shell.
