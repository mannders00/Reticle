# Brand icon

`reticle.svg` is the master — everything else is generated from it.
It mirrors the site's `.brand-mark` (rounded square, radial accent
gradient `#5b8cff → #3d6bff → #2f56d6`, white reticle ring) plus
crosshair ticks and a center dot that read at large sizes.

## Files

| file | purpose |
| --- | --- |
| `reticle.svg` | master vector — edit THIS, regenerate the rest |
| `reticle-1024.png` | master raster (transparent), source for all sets |
| `reticle-{16,32,48,64,128,192,256,512}.png` | general-purpose sizes |
| `favicon.ico` | 16+32+48 multi-size, for the website |
| `apple-touch-icon.png` | 180×180, for the website |

The desktop/platform set (`.icns`, `.ico`, Windows Store logos) lives in
`src-tauri/icons/` and is generated by Tauri, not stored here.

## Regenerating

1. Render the master raster (Chromium renders the SVG pixel-perfectly;
   ImageMagick's SVG delegate does not — don't rasterize the SVG with it):

   ```sh
   # any headless chromium works; see reticle-serve.sh env for the one used in dev
   chromium --headless --screenshot=reticle-1024.png --window-size=1024,1024 \
     --default-background-color=00000000 reticle.svg
   ```

2. Tauri platform icons (Windows .ico, Store logos — these are full-bleed):

   ```sh
   cargo tauri icon assets/icon/reticle-1024.png -o src-tauri/icons
   rm -rf src-tauri/icons/{android,ios}   # desktop-only
   ```

3. macOS `.icns` — REGENERATE SEPARATELY, after step 2 (which overwrites
   it with a full-bleed version). Apple's icon grid wants the squircle at
   824×824 centered in the 1024 canvas (~10% transparent margin) or the
   icon looks oversized in ⌘Tab / the Dock:

   ```sh
   magick reticle-1024.png -resize 824x824 -background none \
     -gravity center -extent 1024x1024 reticle-macos-1024.png
   mkdir -p /tmp/Reticle.iconset
   for s in 16 32 128 256 512; do
     magick reticle-macos-1024.png -resize ${s}x${s} /tmp/Reticle.iconset/icon_${s}x${s}.png
     magick reticle-macos-1024.png -resize $((s*2))x$((s*2)) /tmp/Reticle.iconset/icon_${s}x${s}@2x.png
   done
   iconutil --convert icns /tmp/Reticle.iconset --output ../../src-tauri/icons/icon.icns
   ```

4. Web/general sizes (resizing PNGs with magick is fine — only SVG input is not):

   ```sh
   for s in 16 32 48 64 128 192 256 512; do
     magick reticle-1024.png -resize ${s}x${s} reticle-${s}.png
   done
   magick reticle-1024.png -resize 180x180 apple-touch-icon.png
   magick reticle-16.png reticle-32.png reticle-48.png favicon.ico
   cp favicon.ico apple-touch-icon.png reticle.svg ../../web/
   ```
